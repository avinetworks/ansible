.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is used to configure IPReputationDB object
- more examples at U(https://github.com/avinetworks/devops)


Requirements
------------
The below requirements are needed on the host that executes this module.

- avisdk
